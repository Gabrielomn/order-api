<h2>Orders api</h2>

Hey there, this is an api capable of creating payment orders, and then paying them. Here are some instructions to get you around ;).


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```

## Test

```bash
# e2e tests
$ npm run test:e2e

```

## Documentation

1. Start the application
```bash
$ npm run start

```

2. Go to http://localhost:3000/api/#/
